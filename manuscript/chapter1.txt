# SVN อะไรคือ #

SVN เป็นโปรแกรมตัวหนึ่งที่อยู่ในกลุ่ม SCM หรือ Source Control Management (บริหารจัดการซอร์สโค้ด) มันจะคอยทำหน้าที่เหมือนเครื่องจัดเก็บ Backup แต่ไม่ใช่ Backup อย่างที่เราคุ้นเคย แบบก้อปปี้ทั้งโฟลเดอร์แล้วเปลี่ยนเป็นชื่อใหม่พร้อมระบุวันไว้ ไม่ใช่ Backup ของ SVN จะเหมือน Save Point มากกว่า แบบที่เขียนๆโปรแกรมไป ทดสอบ เห็นว่าโค้ดทำงานได้อย่างถูกต้องแล้ว ก็ทำการ Save Point มันไว้ พร้อมคำอธิบายว่าทำอะไรกับโค้ดลงไป คนจะทำหน้าที่กำหนดว่าจะเก็บ Save Point แล้ว SVN จะทำหน้าที่รวมรวมรายละเอียดว่าแก้ไขโค้ดบรรทัดไหนไปบ้าง แก้อย่างไร เป็นการเพิ่ม หรือแก้ไข หรือลบออก 

เมื่อเขียนโปรแกรมไปนานๆ เราสามารถย้อนกลับมาคลิกดู Save Point จุดต่างๆได้ ไม่เพียงเท่านั้นเราสามารถกำหนดได้ว่า ณ ตำแหน่งตรงไหนคือโปรแกรมเวอร์ชั่น 1.0 ณ ตรงไหนคือโปรแกรมเวอร์ชั่น 2.0 ที่เพิ่มฟีเจอร์ใหม่เข้าไปแล้ว หากลูกค้าไม่ได้ต้องการฟีเจอร์ใหม่ เราก็สามารถ export โค้ดตรงตำแหน่งเวอร์ชั่น 1.0 แล้วส่งขายให้เขาในราคาถูกได้

ที่กล่าวมาข้างต้น เป็นเพียงคุณสมบัติคร่าวๆเท่านั้น SVN ยังมีสิ่งที่มีประโยชน์มหาศาลอีกเพียบ ติดตามได้ในบทต่อๆไป
